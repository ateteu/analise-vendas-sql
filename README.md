# Projeto SQL - Análise de Vendas

## Objetivo
Praticar consultas SQL com um conjunto de dados de vendas e produtos.

## Estrutura dos dados
- `produtos.csv`: lista de produtos com código, nome e preço
- `vendas.csv`: registros de vendas, quantidade, data, país de envio, dentre outras informações

## Como testar
1. Criar banco SQLite (`vendas.db`)
2. Criar tabelas `produtos` e `vendas`
3. Importar os CSVs
4. Rodar as queries do arquivo `queries.sql`

## Insights
- Total de vendas por dia
- Valor total vendido por produto
- Produtos com preço acima da média
- Classificação de produtos por faixa de preço
